О системе

Система общественной печати «Физтех-Печать» предназначена для круглосуточной печати любых документов формата A4. Процесс печати полностью автоматизирован и управляется с помощью веб-сайта.

Вы загружаете документ на сайт, он обработается за несколько секунд. После этого вы сможете выбрать принтер и подтвердить печать. Деньги со счёта спишутся только при успешной печати документа, когда последняя страница выйдет из принтера. Кроме того, отправлять документы вы можете, установив специальный драйвер. Он позволит отправлять документы из любых программ, которые у Вас установлены, и гибко настраивать параметры печати.

Для печати Вам потребуется:

  • Зарегистрироваться

  • Установить драйвер для вашей ОС

    • Windows XP x32/x64
    • Windows Vista x32/x64/Server 2008
    • GNU/Linux/MacOS

    либо использовать кнопку «Добавить файл» в Очереди печати

    В Windows 7, возможно, потребуется изменить драйвер на наш.

  • Пополнить счёт

    • SMS
    • Qiwi
    • WebMoney
    • Visa

Рекомендуем прочесть раздел FAQ перед использованием системы.

Про настройку цветной печати можете прочитать здесь.

При неудовлетворительном качестве отпечатанного документа вы можете вернуть средства на личный счёт.

По любым вопросам о работе системы можно обращаться по почте.
